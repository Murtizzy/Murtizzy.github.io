The primary purpose of web servers is to serve web pages.

What's a web page?
  • A text file in HTML format
  • This is similar to XML
Can I see some HTML?
  • Shift-right-click on a page and then 'View Page Source' or 'Inspect'

What you're looking for after you Shift-right-click
  • The purpose of HTML is to provide content & structure
  • It has got some basic formatting and styling capabilites
  • But if you want cooler styling features check out the next lecture!
